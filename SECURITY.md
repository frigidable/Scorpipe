# Security Policy

Если вы нашли уязвимость или потенциальную проблему безопасности, пожалуйста:

1) **Не создавайте публичный issue** с подробностями эксплуатации.
2) Создайте приватный репорт через GitHub Security Advisories (если включено в репозитории),
   или напишите владельцу проекта через контакт, указанный в README.

Пожалуйста, укажите:
- Версию Scorpio Pipe (`pyproject.toml` или `importlib.metadata.version('scorpio-pipe')`)
- ОС / Python
- Шаги воспроизведения
- Потенциальный impact (что можно сделать злоумышленнику)
- Любые логи/stacktrace (без секретов и токенов)

Мы постараемся ответить как можно быстрее и согласовать раскрытие (responsible disclosure).
